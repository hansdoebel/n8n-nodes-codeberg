# n8n-nodes-codeberg

This is an n8n community node. It lets you use [Codeberg](https://codeberg.org) in your n8n workflows.

Codeberg is a free, open-source Git hosting platform powered by Forgejo — a community-driven alternative to GitHub and GitLab.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/sustainable-use-license/) workflow automation platform.

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation.

## Operations

### Repository
- Create, Delete, Fork, Get, List, Search, Update

### Issue
- Create, Delete, Get, List, Search, Update

### Pull Request
- Create, Get, List, Merge, Update

### Organization
- Create, Delete, Get, List, List Members, Update

### User
- Get, Get by Username, Search

### Comment
- Create, Delete, Get, List, Update

### Label
- Create, Delete, Get, List, Update

### Milestone
- Create, Delete, Get, List, Update

### Release
- Create, Delete, Get, List, Update

### Branch
- Create, Delete, Get, List

### File
- Create, Delete, Get, List, Update

## Credentials

This node supports two authentication methods:

- **API Token** — Generate a personal access token in your Codeberg account under Settings > Applications.
- **OAuth2** — Register an OAuth2 application in your Codeberg account under Settings > Applications.

## Compatibility

Tested with n8n v1.x.
